------------------------------------------------------------------------

نبذة عن المشروع:

تطبيق TareqStore هو متجر إلكتروني تم تطويره باستخدام لغة Java وبيئة
Android Studio مع اعتماد تصميم Material Design 3 الحديث.

يقوم التطبيق بمحاكاة متجر حقيقي يتيح للمستخدم: - تسجيل الدخول وإنشاء
حساب - تصفح التصنيفات والمنتجات - عرض تفاصيل المنتج - إضافة المنتجات إلى
سلة المشتريات - تعديل الكميات وحساب السعر الإجمالي - إتمام عملية الشراء
بعد التحقق من صحة البيانات

------------------------------------------------------------------------

أهداف المشروع:

-   تطبيق التنقل بين عدة شاشات باستخدام Activities
-   استخدام RecyclerView و Adapter لعرض البيانات بشكل ديناميكي
-   تمرير البيانات بين الشاشات باستخدام Parcelable
-   إدارة سلة المشتريات بمنطق برمجي منظم
-   تنفيذ نظام تسجيل دخول باستخدام SharedPreferences
-   تصميم واجهات حديثة ومتناسقة باستخدام Material Design

------------------------------------------------------------------------

التقنيات المستخدمة:

-   Java
-   Android Studio
-   Material Design 3
-   RecyclerView
-   SharedPreferences
-   ConstraintLayout

------------------------------------------------------------------------

الخلاصة:

يعكس هذا المشروع الفهم العملي لأساسيات تطوير تطبيقات الأندرويد من حيث
إدارة الواجهات، تنظيم الكود، التعامل مع البيانات، وبناء تجربة مستخدم
منظمة وسلسة.
